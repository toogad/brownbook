Defines dC/dt for the reactor based on mass balance. 

   Overall mass balance:
dComp/dt == InfFlow / Actvol * (in_comps - mo_comps) + GrowthRate

== (in_comps - mo_comps) / HRT + GrowthRate

Parameters
ttime for use in ODE integration routine, d
mo_compslist of model component for mainstream outlet, mg/L.
volreactor's active volume, m3;
flowreactor's total inflow, m3/d
in_compslist of model compoennts for inlet, mg/L;
Return
